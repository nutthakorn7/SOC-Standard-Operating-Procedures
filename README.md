# SOC Standard Operating Procedures

This repository contains the Standard Operating Procedures (SOPs) for the zcrAI Security Operations Center (SOC). These documents serve as the authoritative reference for the development, deployment, and operational maintenance of the zcrAI platform.

เอกสารนี้รวบรวมระเบียบปฏิบัติมาตรฐาน (SOPs) สำหรับศูนย์ปฏิบัติการความปลอดภัย (SOC) ของ zcrAI เพื่อใช้เป็นแหล่งอ้างอิงหลักสำหรับการพัฒนา การติดตั้งระบบ และการดูแลรักษาระบบ zcrAI

## Contributing / การมีส่วนร่วม

1.  **Consistency**: Ensure all new procedures follow the established Markdown format.
2.  **Verification**: Verify all commands and steps in a safe environment before committing.
3.  **Review**: All changes must be reviewed by the Platform Engineering team.
4.  **Translation**: When updating a document, please update both English and Thai versions to keep them synchronized.
